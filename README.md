# EnuygunTestAutomationProject
Enuygun.com uçuş aramalarını otomatize eden Selenium tabanlı Java framework'ü. Uçuş verilerini çeker, CSV'ye aktarır ve XChart ile dinamik grafikler (Bar Chart, Heat Map) üretir. Fiyat, süre ve aktarma dengesini kuran maliyet-etkin uçuş öneri algoritması ile farklı rotalar için karşılaştırmalı analiz yeteneğine sahiptir.
